What does mandolin do?

mandolin slices up gherkin to make it more versatile

Why?

gherkin is great but scenarios, particularly scenario outlines, have their data embedded. What if data needs to change over time or needs to come from another data source before a test is run?

Mandolin has two functions:

mandolin can do the following:

  • strip out tables and place each one in a pipe separated file
  • merge a feature file (containing table placeholders) with a set of pipe separated data files
