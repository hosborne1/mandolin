Mandolin

slices up Gherkin to make it more versatile

Download this project as a .zip file Download this project as a tar.gz file

What is mandolin?

mandolin is a utility for slicing up Gherkin files which are used as part of Behavior Driven Development.

Why?

Gherkin is great but one of its main strengths, having a simple clear editable-by-anyone format, can also present issues with scalability and maintenance when reusing the scenarios for regression testing.

By providing a way to strip out and then re-merge tables to and from separate pipe delimited text files, the data can be managed in any way you like

mandolin can do the following:

  • strip out tables and place each one in a pipe separated file
  • merge a feature file (containing table placeholders) with a set of pipe separated data files
